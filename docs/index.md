# Velox Linux Handbook

**Shape It. Race It. Own It.**

Velox Linux is a custom Arch Linux distribution built for creators, gamers, and power users. It ships with KDE Plasma, a curated package selection, the performance-tuned `linux-velox` kernel, and a streamlined installer that gets you running in minutes.

This handbook covers everything from your first boot to building the ISO yourself.

---

## What's in the box

- **KDE Plasma 6** — fully themed with the Velox dark palette
- **linux-velox kernel** — CachyOS-based, locked to the `performance` CPU governor
- **Calamares installer** — graphical, ext4, with automatic NVIDIA detection
- **velox-welcome** — first-run app for packages, snapshots, gaming, and more
- **Flatpak + paru + chaotic-aur** — the full Arch package ecosystem out of the box
